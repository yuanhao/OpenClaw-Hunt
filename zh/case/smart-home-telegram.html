通过 Telegram 控制智能家居

通过聊天控制灯光、恒温器、门锁。自然语言家庭自动化。

命令

  • "关掉楼下所有灯"
  • "把恒温器调到 72°F"
  • "锁上前门"
  • "车库门关了吗？"
  • "把客厅灯光调到 30%"

工作原理

  1. 通过 Telegram 发送自然语言命令
  2. 代理解析意图和目标设备
  3. 调用 Home Assistant API
  4. 用状态更新确认操作

集成

支持：

  • Home Assistant
  • Philips Hue
  • Nest 恒温器
  • August 门锁
  • MyQ 车库门
  • 以及 1000+ 其他设备

安全

  • 白名单特定 Telegram 用户
  • 敏感操作需确认（解锁、车库）
  • 所有命令的审计日志
