# Content Sourcing Engine

Automated newsletter research across Reddit, GitHub, YouTube, Discord, and blogs with multi-source aggregation.

## The Workflow

```
Reddit → GitHub → YouTube → Discord → Blogs
   ↓         ↓         ↓         ↓        ↓
   └─────────┴─────────┴─────────┴────────┘
                    ↓
            Content Aggregation
                    ↓
            Summarization & Categorization
                    ↓
            Newsletter Drafting
```

## Data Sources

- **Reddit** - Subreddit monitoring, trending discussions
- **GitHub** - New repositories, trending projects
- **YouTube** - New videos from subscribed channels
- **Discord** - Community discussions, announcements
- **Blogs** - RSS feeds, tech blogs, newsletters

## Output

- Curated content summaries
- Auto-categorized by topic
- Ready-to-edit newsletter drafts
- Cross-platform publishing ready

## Time Saved

- **Manual research**: 4-6 hours/day
- **With OpenClaw**: 15 minutes review
- **Net savings**: 90%+ time reduction

# 内容采集引擎

跨 Reddit、GitHub、YouTube、Discord 和博客的自动化新闻通讯研究，多源聚合。

## 工作流

```
Reddit → GitHub → YouTube → Discord → 博客
   ↓         ↓         ↓         ↓        ↓
   └─────────┴─────────┴─────────┴────────┘
                    ↓
              内容聚合
                    ↓
              摘要与分类
                    ↓
              新闻通讯起草
```

## 数据源

- **Reddit** - 子版块监控、热门讨论
- **GitHub** - 新仓库、热门项目
- **YouTube** - 订阅频道的新视频
- **Discord** - 社区讨论、公告
- **博客** - RSS 订阅、技术博客、新闻通讯

## 输出

- 精选内容摘要
- 自动按主题分类
- 可编辑的新闻通讯草稿
- 跨平台发布就绪

## 节省时间

- **手动研究**: 每天 4-6 小时
- **使用 OpenClaw**: 15 分钟审查
