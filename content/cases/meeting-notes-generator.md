# Meeting Notes Generator

Paste raw notes, get structured minutes. Transform chaos into professional documentation.

## Input Example

```
john said we need to fix the login bug by friday
sarah will handle the design review
budget approved for q3 marketing
```

## Output

```
## Meeting Minutes - [Date]

### Action Items
- **@john** - Fix login bug (Due: Friday)
- **@sarah** - Handle design review

### Decisions
- Q3 marketing budget approved

### Attendees
John, Sarah
```

## Features

- **Action item extraction** - Who does what by when
- **Decision logging** - Key outcomes captured
- **Attendee tracking** - Auto-detect participants
- **Follow-up reminders** - Calendar integration

## 中文

# 会议记录生成器

粘贴原始笔记，获取结构化纪要。将混乱转化为专业文档。

## 输入示例

```
John 说我们需要在周五前修复登录 bug
Sarah 将处理设计审查
Q3 营销预算已批准
```

## 输出

```
## 会议纪要 - [日期]

### 行动项
- **@john** - 修复登录 bug（截止：周五）
- **@sarah** - 处理设计审查

### 决策
- Q3 营销预算已批准

### 与会者
John、Sarah
```

## 功能

- **行动项提取** - 谁做什么，什么时候
- **决策记录** - 捕获关键结果
- **与会者跟踪** - 自动检测参与者
- **跟进提醒** - 日历集成
