# Email Newsletter to Podcast

Turn newsletters into audio for your commute. Never miss important content because you're driving or exercising.

## How It Works

1. Agent monitors your inbox for newsletters
2. Extracts key content and summaries
3. Converts text to speech using TTS
4. Delivers audio file to your podcast app

## Setup

```
Every morning at 7 AM:
- Check inbox for newsletters from [list]
- Summarize each newsletter
- Convert to audio using TTS
- Upload to my podcast feed
- Send notification when ready
```

## Benefits

- **Time saved**: 30+ minutes daily reading time
- **Multitasking**: Learn while commuting, exercising, cooking
- **Better retention**: Audio often sticks better than skimming

## 中文

# 邮件通讯转播客

将新闻通讯转换为音频，方便通勤时收听。再也不会因为开车或运动而错过重要内容。

## 工作原理

1. 代理监控收件箱中的新闻通讯
2. 提取关键内容和摘要
3. 使用 TTS 转换为语音
4. 将音频文件发送到你的播客应用

## 设置

```
每天早上 7 点：
- 检查来自 [列表] 的新闻通讯
- 总结每个新闻通讯
- 使用 TTS 转换为音频
- 上传到我的播客源
- 准备好后发送通知
```

## 好处

- **节省时间**: 每天节省 30+ 分钟阅读时间
- **多任务处理**: 通勤、运动、做饭时学习
- **更好记忆**: 音频往往比略读更容易记住
