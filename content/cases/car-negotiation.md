# Car Negotiation Bot

Agent scrapes dealer inventories, fills contact forms, plays dealers against each other with competing quotes.

## The Strategy

1. **Scrape inventories** - Multiple dealerships
2. **Fill contact forms** - Automated outreach
3. **Collect quotes** - Get competing offers
4. **Play dealers against each other** - Use quotes to negotiate

## Result

**$4,200 below sticker price**

## How It Works

The agent automates the tedious process of contacting multiple dealers and collecting quotes, then uses those quotes to negotiate better deals.
