# Three-Tier Memory System

Long-term + daily + project layers. Organized knowledge that scales.

## The Three Tiers

### 1. Long-term Memory
- Core knowledge about you
- Preferences, goals, relationships
- Rarely changes

### 2. Daily Memory
- Today's context
- Current tasks, meetings, decisions
- Refreshed each morning

### 3. Project Memory
- Active project context
- Code, docs, decisions
- Archived when project ends

## How It Works

1. **Morning sync** - Load daily + relevant project layers
2. **During day** - Update all three tiers as needed
3. **Evening archive** - Compress daily into long-term
4. **Project end** - Archive project memory

## Benefits

- **Context efficiency** - Only load what's needed
- **Scalable** - Handle 100+ projects without bloat
- **Organized** - Clear separation of concerns

## 中文

# 三层记忆系统

长期 + 每日 + 项目层。可扩展的有序知识。

## 三层结构

### 1. 长期记忆
- 关于你的核心知识
- 偏好、目标、关系
- 很少变化

### 2. 每日记忆
- 今日上下文
- 当前任务、会议、决策
- 每天早上刷新

### 3. 项目记忆
- 活跃项目上下文
- 代码、文档、决策
- 项目结束时归档

## 工作原理

1. **早间同步** - 加载每日 + 相关项目层
2. **白天** - 按需更新三层
3. **晚间归档** - 将每日压缩到长期
4. **项目结束** - 归档项目记忆

## 好处

- **上下文效率** - 只加载需要的
- **可扩展** - 处理 100+ 项目而不膨胀
- **有序** - 关注点清晰分离
