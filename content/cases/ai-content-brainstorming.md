# AI Content Brainstorming Agent

Automated content idea generation based on industry trends and competitor analysis.

## Overview

Running out of content ideas? OpenClaw monitors industry news, analyzes trending topics, checks competitor publications, and suggests content angles based on what's getting attention.

## How It Works

- Monitors RSS feeds from industry sites
- Checks social media for trending topics in your niche
- Analyzes community channels for common questions
- Reviews competitor content
- Generates tailored content suggestions

## Example Output

You wake up to a message listing five content ideas:
- "Write about the new API security standards announced yesterday – trending in your industry."
- "Create a comparison post between Tool A and Tool B – your competitor just published one and got good engagement."
- "Tutorial on solving X problem – three people asked about this in your Discord this week."

## Benefits

- Never stare at a blank page wondering what to write
- Data-driven content ideas based on actual trends
- Still decide what to write and how to write it

## 中文

# AI 内容创意生成代理

基于行业趋势和竞品分析自动生成内容创意。

## 概述

没内容灵感了？OpenClaw 监控行业新闻、分析热门话题、检查竞品发布，并根据热点推荐内容角度。

## 工作原理

- 监控行业网站的 RSS 订阅
- 检查你所在领域的社交媒体热门话题
- 分析社区频道的常见问题
- 审查竞品内容
- 生成定制化的内容建议

## 示例输出

醒来看到一条消息，列出五个内容创意：
- "写关于昨天发布的新 API 安全标准——在你的行业 trending"
- "创建 Tool A 和 Tool B 的对比文章——你的竞品刚发了一篇，互动很好"
- "解决 X 问题的教程——这周有三个人在你的 Discord 问了这个问题"

## 优势

- 再也不用盯着空白页面想写什么
- 基于实际趋势的数据驱动内容创意
- 仍然由你决定写什么、怎么写
