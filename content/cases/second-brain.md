# Second Brain with Semantic Search

Text anything to your bot to remember it, then search through all your memories with vector-powered semantic search.

## How It Works

### Capture
```
You: "Remember that article about vector databases I read last week"
Bot: ✓ Saved to memory
```

### Search
```
You: "What was that thing about embedding similarity?"
Bot: Found 3 relevant memories...
```

## Features

- **Natural Language Input** - Text, voice, or image
- **Auto-Categorization** - AI tags and organizes
- **Semantic Search** - Find by meaning, not just keywords
- **Hybrid Retrieval** - Combines keyword + vector search
- **Next.js Dashboard** - Visual memory browser

## Technical Stack

- **Vector DB** - Chroma, Pinecone, or Weaviate
- **Embeddings** - OpenAI, local, or custom models
- **Sync** - Auto-sync with markdown memory files
- **API** - RESTful access to all memories

## Use Cases

- Research note-taking
- Meeting notes and decisions
- Article and book summaries
- Idea capture and development
- Personal knowledge management

## 中文

# 带语义搜索的第二大脑

给机器人发任何内容让它记住，然后通过向量驱动的语义搜索检索所有记忆。

## 工作原理

### 捕获
```
你: "记住我上周读的那篇关于向量数据库的文章"
机器人: ✓ 已保存到记忆
```

### 搜索
```
你: "那个关于嵌入相似性的东西是什么？"
机器人: 找到 3 条相关记忆...
```

## 功能

- **自然语言输入** - 文本、语音或图片
- **自动分类** - AI 标签和整理
- **语义搜索** - 按含义搜索，不只是关键词
- **混合检索** - 结合关键词 + 向量搜索
- **Next.js 仪表板** - 可视化记忆浏览器

## 技术栈

- **向量数据库** - Chroma、Pinecone 或 Weaviate
- **嵌入模型** - OpenAI、本地或自定义模型
- **同步** - 自动同步 markdown 记忆文件
- **API** - 所有记忆的 RESTful 访问

## 使用场景

- 研究笔记
- 会议记录和决策
- 文章和书籍摘要
- 想法捕获和发展
- 个人知识管理
