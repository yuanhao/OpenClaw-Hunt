# Daily Briefing in 5 Minutes

Morning digest with weather, calendar, news, and priorities. Saves 30+ minutes daily.

## Setup

```
Every morning at 7:00 AM, send me a briefing with:
- Weather
- First 3 calendar events
- Top 3 headlines
- Open tasks
- One priority for today
```

## What You Get

- **Weather** - Local forecast for the day
- **Calendar** - First 3 events with time and location
- **News** - Top 3 headlines from your preferred sources
- **Tasks** - Open tasks from your task manager
- **Priority** - One AI-suggested priority based on deadlines

## Time Saved

30+ minutes daily by eliminating manual checking of multiple apps.

# 5 分钟设置每日简报

早间摘要包含天气、日程、新闻和优先事项。每天节省 30+ 分钟。

## 设置

```
每天早上 7:00 发送简报：
- 天气
- 前 3 个日程事件
- 前 3 条新闻头条
- 待办任务
- 今日优先事项
```

## 内容

- **天气** - 当天本地预报
- **日程** - 前 3 个事件的时间和地点
- **新闻** - 首选来源的前 3 条头条
- **任务** - 任务管理器中的待办事项
- **优先事项** - 基于截止日期的 AI 建议优先事项

## 节省时间

通过消除手动检查多个应用，每天节省 30+ 分钟。
