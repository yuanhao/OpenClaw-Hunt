# Shopify → Romanian Post Automation

Full e-commerce pipeline: read order details from Shopify, open courier website, fill AWB fields, print label. All via Kimi 2.5 with browser automation.

## The Workflow

1. **Read order** from Shopify (name, address, phone, email)
2. **Open** Romanian Post (Poșta Română) AWB website
3. **Auto-fill** all shipping fields
4. **Ask for confirmation** then finalize/print label

## Challenges

- Chrome extension frequently disconnects
- Sessions fail mid-process
- Browser automation reliability issues

## Model Used

**Kimi 2.5** - For browser automation and form filling

## Quote

> "Even partial automation would already save me a lot of time."

# Shopify → 罗马尼亚邮政自动化

完整电商流程：从 Shopify 读取订单详情，打开快递公司网站，填写 AWB 字段，打印标签。全部通过 Kimi 2.5 浏览器自动化完成。

## 工作流程

1. **从 Shopify 读取订单**（姓名、地址、电话、邮箱）
2. **打开** 罗马尼亚邮政（Poșta Română）AWB 网站
3. **自动填写** 所有发货字段
4. **请求确认** 然后完成/打印标签

## 挑战

- Chrome 扩展频繁断开连接
- 会话中途失败
- 浏览器自动化可靠性问题

## 使用的模型

**Kimi 2.5** - 用于浏览器自动化和表单填写
