# Daily Morning Briefing

Weather + calendar + news in one Telegram message. Start your day informed.

## What You Get

- **Weather** - Today's forecast with outfit suggestions
- **Calendar** - First 3 events with prep time
- **News** - Top headlines from your sources
- **Tasks** - Overdue and today's priorities

## Setup

```
Every morning at 7:30 AM:
- Check weather for my location
- List first 3 calendar events today
- Fetch top 5 headlines from [sources]
- Show overdue tasks
- Send everything in one Telegram message
```

## 中文

# 每日早间简报

天气 + 日程 + 新闻，一条 Telegram 消息。开启知情的一天。

## 内容

- **天气** - 今日预报及穿搭建议
- **日程** - 前 3 个事件及准备时间
- **新闻** - 你关注来源的热门头条
- **任务** - 逾期和今日优先级

## 设置

```
每天早上 7:30：
- 查询我所在位置的天气
- 列出今天前 3 个日程事件
- 从 [来源] 获取前 5 条头条
- 显示逾期任务
- 将所有内容发送到一条 Telegram 消息
```
